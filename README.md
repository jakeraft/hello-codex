# hello-codex

Tutorial team for [clier](https://github.com/jakeraft/clier). Single-agent
codex demo. Wired as a subteam of `jakeraft/hello-clier`.

The team source lives at [`teams/hello-codex/`](teams/hello-codex/) so
clier's `git_subpath` mechanism gets exercised on this repo (the sibling
`jakeraft/hello-clier` puts everything at the repo root for contrast).

Contents (under `teams/hello-codex/`):
- `AGENTS.md` — codex instruction.
- `.codex/skills/self-intro/SKILL.md` — invoked when the user asks "who are you?".
